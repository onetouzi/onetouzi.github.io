## 🚀 项目经历

### 1. YOLOv5 目标检测模型的 TensorRT 推理优化
*   **时间**：2026.02 - 2026.05
*   **技术栈**：OpenCV + CUDA + TensorRT + PyTorch + C++
*   **主要工作**：
    1.  **网络手工搭建**：使用 TensorRT Network Definition API 手工搭建完整的 YOLOv5 网络，替代 ONNX 导出，以获得更细粒度的图优化控制。
    2.  **CUDA 算子开发**：编写 CUDA 自定义算子（YoloLayer Plugin）封装多尺度检测头的 anchor 解码与边框回归逻辑，解决标准算子无法表达的定制计算。
    3.  **GPU 并行预处理**：实现 GPU 端 CUDA 并行预处理（letterbox resize + 归一化），将图像预处理从 CPU 迁移至 GPU，消除数据搬运瓶颈。
    4.  **INT8 量化**：基于 Entropy Calibrator 实现 INT8 量化推理，完成校准数据采集与精度验证。

### 2. 电商券管平台（高并发秒杀系统）
*   **时间**：2026.01 - 2026.03
*   **核心技术**：SpringBoot + RocketMQ + ShardingSphere + Redis + MySQL
*   **主要工作**：
    1.  **大数据量流式解析**：使用 **EasyExcel** 解析百万量级用户优惠券推送 Excel 文件，结合流式处理避免因大文件导致的内存溢出（OOM）问题。
    2.  **高并发缓存设计**：采用**布隆过滤器**、缓存空值和**分布式锁**等方案，解决秒杀与高并发场景下可能导致的缓存击穿和穿透问题。
    3.  **工程重构与设计模式**：采用**责任链模式**重构参数校验逻辑，使验证代码高内聚、低耦合，严格保障开闭原则。
    4.  **精准幂等与高并发重试**：通过创建用户优惠券表的**唯一索引**强保障发放幂等性。结合**乐观自旋机制**反复尝试，确保发放优惠券时不漏发、不多发、不超发。
